Conception - CAO

CAD: Computed aided design / CAO: Conception assisté par ordinateur

Il existe de nombreux logiciels de conception 3D, mais on peut distinguer 3 grandes catégories. A noter qu'il existe aussi des logiciels "hybrides" à la frontière des ces catégories.

Les logiciels paramétriques

Ces logiciels permettent de spécifier des entités géométriques (ligne, courbe, surface) ainsi que les relations entre ces entités. ces relations sont comparables ã  des fonctions dont l'utilisateur ajuste les paramètres d'obtenir le résultat escompté (cotation, angle, rotation, duplication…). par exemple pour faire un cube, l'utilisateur doit d'abord dessiner un carré (surface délimité par 4 segments égaux et orthogonaux) puis y appliquer une extrusion de longueur égale au segment formant le carré. ainsi, le paramètre de la fonction d'extrusion est le mãªme que le paramètre de fonction definissant le carré. en changeant un paramètre, l'utilisateur change le volume. imaginons que ce cube aient été dupliqué (fonction répétition), alors toutes les répétitions de ce cube seront également modifiée.

Mesh Modeling

Ce dernier type de modélisation se destine plus à un usage "artistique" comme la création de personnages ou de textures. Le concepteur a directement la main sur le maillage (mesh) qui compose l'objet, agissant comme un sculpteur sur l'objet. On parle alors de mesh modeling.

Installation

Avant de toutes choses, Solidworks est uniquement compatible avec Windows et ne fonctionnera pas sur une machine virtuelle…
Voici la procédure pour installer la version étudiante de Solidworks (SDK 2022-2023). Cette version est gratuite et valable 1 an:
0 - Désinstaller tous les Solidworks avant d’installer une nouvelle version.
1 – Accéder à la page de téléchargement. Attention, il sera nécessaire de télécharger le hotfix, après installation du logiciel, voir le message en haut de page avant d’indiquer votre nom et adresse mail.
2 – Cliquez sur OUI (question « J’ai déjà un numéro de série »)
4 – Choisissez la version 2022-2023
5 – Numéro de série Solidworks SDK 2022 : 90200054428870073GXSVZBC
6- Télécharger le fichier exécutable.
7- Exécuter le fichier, le « SLDIM » se lance et faites un téléchargement complet tous les fichiers
8- Lancer l’installation une fois que vous avez tous les fichiers sur votre poste à partir de setup.exe
Si vous n’activez pas la licence, vous n’aurez qu’une licence de 30 jours. Cette licence expirera 365 jours après la date d’activation ou le 31 juillet 2024, selon la première éventualité.
